Evidence Vault (UI)

This project implements the Evidence Vault list view for SentryLink Comply Phase A,
focused on the factory user experience.

## Implemented
- Evidence Vault table using mocked JSON data
- Search by document name (prefix-based)
- Filters by document type and status
- Bulk selection with “Add to Pack” indicator
- Reusable StatusChip component
- Local state only (no backend)

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS

## Notes
This implementation intentionally focuses on the core Evidence Vault behavior,
keeping the scope aligned with the task instructions.

## Run locally
npm install
npm run dev
